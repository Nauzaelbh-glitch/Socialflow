Design a modern SaaS dashboard overview page for SocialFlow, a social media management platform.

**DESIGN SYSTEM:**
- Primary color: Emerald green (#10B981)
- Accent: Teal (#14B8A6)
- Neutral grays: from #F9FAFB to #111827
- Semantic colors: Success #22C55E, Warning #F59E0B, Error #EF4444, Info #3B82F6
- Font: Nunito (Google Fonts), fallback sans-serif
- Border radius: 8px-16px (rounded-lg to rounded-2xl)
- Shadows: subtle (shadow-sm to shadow-lg)
- Layout: Sidebar (250px) + Main content area
- Style: Modern, friendly, professional SaaS dashboard

**Page Structure:**
1. Header with logo, search bar, notifications bell, and user avatar
2. Sidebar navigation (collapsible): Dashboard, Empresas, Cuentas Sociales, Publicaciones, Calendario, Analytics, Configuración
3. Main content with:
   - Welcome message with date
   - 4 metric cards (Total Posts, Engagement Rate, Reach, Active Accounts)
   - Recent activity feed
   - Quick actions panel (Nueva Publicación, Conectar Cuenta)
   - Platform status cards showing connected social accounts (Facebook, Instagram, LinkedIn, Twitter, TikTok)
   - Mini chart showing engagement trend

**Visual Elements:**
- Cards with white background, subtle shadow, rounded corners
- Social platform icons with brand colors (Facebook blue, Instagram gradient, LinkedIn blue, Twitter black, TikTok cyan)
- Clean data visualization
- Status badges for account connectivity
